# Skill: Quality, Performance, Accessibility

## Цель
Сайт быстрый, доступный, без регрессий.

## Performance
- next/image everywhere
- минимальный JS на клиенте
- избегать тяжёлых libs
- правильные responsive sizes

## Accessibility
- семантические теги
- keyboard navigation
- focus states
- aria labels для иконок/кнопок без текста

## UX safeguards
- skeleton/loading states для галерей/форм
- понятные empty states (если нет кейсов)

## DoD (Release checklist)
- Lighthouse без критических проблем
- mobile navigation не ломается
- формы не падают
- изображения не “скачут” (фикс aspect ratio)
