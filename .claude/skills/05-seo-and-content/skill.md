# Skill: SEO, Copy & Content Model

## Цель
Сделать сайт, который индексируется, красиво шарится, и повышает конверсию за счёт структуры и сильного копирайта.

## SEO-минимум (обязательно)
- Metadata API на страницах.
- OpenGraph + Twitter cards.
- Sitemap + robots.
- ЧПУ: /portfolio/[slug], /services, /pricing, /b2b/...

## Контент-правила (позиционирование)
- Всегда подчёркиваем: реализуемость, инженерка, экономика, этапы, границы.
- Убираем “поэзию ради поэзии”, пишем конкретно (deliverables, сроки, правки).
- В FAQ снимаем возражения: цена, сроки, правки, надзор, состав проекта.

## Контент-модель
- Projects (кейсы) — mdx.
- Packages — data file (ts/json).
- FAQ — mdx/ts.

## JSON-LD (желательно)
- LocalBusiness/Organization
- Service
- FAQPage

## DoD
- Каждая ключевая страница имеет уникальный title/description.
- Кейсы дают органический трафик (структура, заголовки, alt).
- CTA встроены в контент, а не “в конце где-то”.
