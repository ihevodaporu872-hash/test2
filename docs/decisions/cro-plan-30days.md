# 30-дневный план CRO (Conversion Rate Optimization)

> Дата создания: 2026-02-07
> Цель: увеличить конверсию сайта студии дизайна интерьера на 15-25% за 30 дней

---

## Общий подход

- Каждый тест запускается минимум на 7 дней (или до 100+ конверсий на вариант)
- Метрики отслеживаются через Яндекс.Метрику (цели) и встроенную аналитику
- Приоритет определяется по формуле: потенциальный эффект x простота внедрения
- Одновременно запускаем не более 2 тестов, чтобы не смешивать результаты

---

## Гипотезы A/B тестов

### 1. Заголовок Hero-блока — конкретика vs эмоция

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: «Дизайн интерьера с инженерной точностью» vs Вариант B: «Квартира, в которую хочется возвращаться — за 60 дней» |
| **Ожидаемый эффект** | +10-15% кликов по CTA в hero |
| **Метрика** | CTR кнопки «Рассчитать стоимость» в hero-блоке |
| **Приоритет** | **Высокий** |

### 2. Цвет и текст основной CTA-кнопки

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: зелёная кнопка «Рассчитать стоимость» vs Вариант B: оранжевая кнопка «Узнать цену за 2 минуты» |
| **Ожидаемый эффект** | +5-10% кликов по CTA |
| **Метрика** | CTR основной CTA на всех экранах |
| **Приоритет** | **Высокий** |

### 3. Расположение CTA — после hero vs плавающая кнопка

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: CTA только в секциях vs Вариант B: дополнительная плавающая кнопка внизу экрана на мобильных |
| **Ожидаемый эффект** | +8-12% заявок с мобильных |
| **Метрика** | Количество открытий формы заявки (мобильный трафик) |
| **Приоритет** | **Высокий** |

### 4. Карточки пакетов — горизонтальные vs вертикальные

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: текущий вертикальный grid 3 колонки vs Вариант B: горизонтальные карточки с акцентом на «что входит» |
| **Ожидаемый эффект** | +5-8% кликов по кнопке «Подробнее» на карточке |
| **Метрика** | CTR кнопок карточек пакетов |
| **Приоритет** | **Средний** |

### 5. Социальное доказательство — отзывы выше vs ниже пакетов

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: отзывы после блока пакетов vs Вариант B: отзывы сразу после hero (до пакетов) |
| **Ожидаемый эффект** | +7-12% доскроллов до формы заявки |
| **Метрика** | Scroll depth до формы + конверсия в заявку |
| **Приоритет** | **Высокий** |

### 6. FAQ — свёрнутый vs развёрнутый первый вопрос

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: все вопросы свёрнуты vs Вариант B: первые 3 вопроса развёрнуты по умолчанию |
| **Ожидаемый эффект** | +3-5% взаимодействий с FAQ, снижение bounce rate на странице |
| **Метрика** | Время на странице + клики в секции FAQ |
| **Приоритет** | **Низкий** |

### 7. Длина формы — 5 полей vs 3 поля

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: имя + телефон + email + площадь + комментарий vs Вариант B: имя + телефон + площадь (минимум) |
| **Ожидаемый эффект** | +15-25% завершений формы |
| **Метрика** | Conversion rate формы (открытие → отправка) |
| **Приоритет** | **Высокий** |

### 8. Отображение цены — «от X руб.» vs «от X руб./м²»

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: цена общая «от 150 000 руб.» vs Вариант B: цена за м² «от 2 500 руб./м²» |
| **Ожидаемый эффект** | +5-10% кликов по пакету, снижение отказов |
| **Метрика** | CTR карточки пакета + bounce rate на странице услуг |
| **Приоритет** | **Средний** |

### 9. Бейдж «Популярный» на среднем пакете

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: все пакеты одинаковые vs Вариант B: средний пакет выделен бейджем «Популярный выбор» и увеличен |
| **Ожидаемый эффект** | +10-15% выбора среднего пакета (увеличение среднего чека) |
| **Метрика** | Распределение кликов по пакетам |
| **Приоритет** | **Средний** |

### 10. Блок «Как мы работаем» — с таймлайном vs без

| Параметр | Значение |
|---|---|
| **Что тестируем** | Вариант A: текстовый список этапов vs Вариант B: визуальный таймлайн с иконками и сроками |
| **Ожидаемый эффект** | +5-8% доскроллов до формы, повышение доверия |
| **Метрика** | Scroll depth + время на странице |
| **Приоритет** | **Средний** |

## Таймлайн на 30 дней

### Неделя 1 (дни 1-7): Подготовка + быстрые победы

| День | Действие |
|---|---|
| 1-2 | Настройка целей в Яндекс.Метрике, подключение A/B-инструмента |
| 3 | Запуск теста #7 (длина формы) — самый высокий потенциальный эффект |
| 4 | Запуск теста #1 (заголовок hero) |
| 5-7 | Сбор данных, промежуточный анализ |

### Неделя 2 (дни 8-14): Основные тесты

| День | Действие |
|---|---|
| 8 | Анализ промежуточных результатов тестов #7 и #1 |
| 9 | Запуск теста #2 (цвет/текст CTA) |
| 10 | Запуск теста #5 (расположение соц. доказательств) |
| 11-14 | Сбор данных, фиксация победителя теста #7 (если статзначимость достигнута) |

### Неделя 3 (дни 15-21): Углубление

| День | Действие |
|---|---|
| 15 | Фиксация победителей тестов #1, #2 |
| 16 | Запуск теста #3 (плавающая CTA на мобильных) |
| 17 | Запуск теста #9 (бейдж «Популярный») |
| 18-21 | Сбор данных, анализ |

### Неделя 4 (дни 22-30): Оптимизация + итоги

| День | Действие |
|---|---|
| 22 | Фиксация победителей тестов #3, #5, #9 |
| 23 | Запуск тестов #4 (карточки пакетов) и #8 (отображение цены) |
| 24-27 | Сбор данных |
| 28 | Запуск теста #10 (таймлайн) и #6 (FAQ) — низкий приоритет, на финал |
| 29-30 | **Итоговый отчёт**: суммарный прирост конверсии, план на следующий месяц |

---

## Ожидаемый результат

- Суммарный прирост конверсии: **15-25%** (кумулятивный эффект от нескольких улучшений)
- Наибольший эффект ожидается от: сокращения формы (#7), заголовка hero (#1), плавающей CTA (#3)
- По итогам 30 дней формируется обновлённый план на следующий цикл оптимизации
