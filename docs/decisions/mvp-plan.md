## 2. Двухнедельный план MVP (Sprint 1-2)

**Цель MVP**: Работающий сайт с 5 основными страницами, русским контентом, SEO, формой заявки, деплой на Vercel.

### Неделя 1 — Фундамент контента и главная страница

| День | Задача | Результат |
|------|--------|-----------|
| Пн | Русификация data-файлов (projects, services, team, testimonials) | Все данные на русском, реальные фото вместо placeholder |
| Пн | Создание `content/` с текстами страниц | Тексты Hero, About, Services, FAQ |
| Вт | Секция Hero (главная) | Анимированный Hero с фоновым изображением, заголовком, CTA |
| Вт | Секция Services Preview (главная) | Карточки 3 услуг с иконками, ссылка «Все услуги» |
| Ср | Секция Featured Projects (главная) | Сетка/карусель 3 избранных проектов |
| Ср | Секция Testimonials (главная) | Слайдер отзывов (Embla/Swiper) |
| Чт | Секция Process (главная) | 4-шаговый визуальный процесс |
| Чт | Секция CTA (главная) | Финальный CTA-блок с формой/кнопкой |
| Пт | Сборка главной из секций | `page.tsx` = Hero + Services + Projects + Process + Testimonials + CTA |
| Пт | Header + Footer с навигацией | Меню, логотип, контакты в футере, мобильное меню |

**Зависимости недели 1**:
```
content/ + data/ (русификация) → секции → сборка page.tsx
layout компоненты (header, footer) → параллельно с секциями
```

### Неделя 2 — Внутренние страницы + SEO + Деплой

| День | Задача | Результат |
|------|--------|-----------|
| Пн | Страница `/services` | Карточки услуг с описаниями, CTA |
| Пн | Страница `/about` | Команда, миссия, ценности |
| Вт | Страница `/portfolio` | Фильтруемая сетка проектов |
| Вт | Страница `/portfolio/[slug]` | Детальная страница: галерея, до/после, описание |
| Ср | Страница `/contact` | Форма (react-hook-form + zod), карта, контакты |
| Ср | Страница `/faq` | Аккордеон с вопросами-ответами |
| Чт | SEO: metadata, OG-изображения, JSON-LD | Все страницы с мета-тегами, robots.txt, sitemap |
| Чт | Перфоманс: lazy-load, Image optimization | LCP < 2.5s, CLS < 0.1 |
| Пт | Деплой на Vercel | Production URL, проверка всех страниц |
| Пт | Smoke-тестирование | Все маршруты отвечают 200, формы работают |

**Зависимости недели 2**:
```
Русский контент (неделя 1) → страницы услуг, о нас
data/projects.ts (русификация) → portfolio, portfolio/[slug]
Форма contact → может работать параллельно (нет зависимостей)
SEO → после сборки всех страниц
Деплой → после SEO + перфоманс
```

### Критерии готовности MVP

- [ ] 7 страниц работают и заполнены русским контентом
- [ ] Главная страница содержит минимум 5 секций
- [ ] Контактная форма отправляет данные (email/webhook)
- [ ] Мобильная адаптация на всех страницах
- [ ] PageSpeed Insights: Performance > 85, SEO > 90
- [ ] Деплой на production URL

## 5. Риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Контент на русском не готов вовремя | Высокая | Блокирует всё | Писать контент параллельно с разработкой, начать с текстов главной |
| Фотографии проектов не предоставлены | Средняя | Портфолио выглядит пустым | Использовать photo_1..10 как временные, запросить реальные фото у студии |
| Scope creep (расширение требований) | Высокая | Срыв сроков | Строгий бэклог с приоритетами, новые задачи — только в Sprint 5-6 |
| Проблемы с перфомансом из-за тяжёлых анимаций | Средняя | Плохой PageSpeed | Профилировать рано, GSAP fallback на CSS |
| CRM-интеграция сложнее ожидаемого | Низкая | Задержка лидогенерации | Начать с email-уведомлений, CRM — отдельным этапом |
